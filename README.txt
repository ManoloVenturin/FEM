# FEM Projects in Julia

This repository contains introductory projects and example code for the Finite Element Method (FEM), written in [Julia](https://julialang.org/).

It is intended as educational material for anyone interested in learning the basics of FEM using Julia, with simple examples and well-commented code.

- Julia source code
- Example scripts
